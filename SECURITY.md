# Security Policy

## Supported Versions

`streamcache` is distributed as a Go module. Security fixes are applied to the
latest released version; please stay current with the most recent release.

| Version  | Supported          |
| -------- | ------------------ |
| latest   | :white_check_mark: |
| < latest | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** — do not open a public
issue or pull request for a suspected vulnerability.

Use GitHub's private vulnerability reporting for this repository:

1. Go to the
   [Security tab](https://github.com/neilotoole/streamcache/security).
2. Click **Report a vulnerability**.
3. Include the affected version, reproduction steps, and the impact you've
   observed.

You can expect an initial acknowledgement within a few days. Once a report is
triaged, a fix will be developed and disclosure coordinated through a
[GitHub Security Advisory](https://github.com/neilotoole/streamcache/security/advisories),
crediting the reporter unless anonymity is requested.
